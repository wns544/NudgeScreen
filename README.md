# Todo Lock

갤럭시 잠금화면 위에 할일 목록을 먼저 보여주는 Android 앱입니다. 앱이 지문인식을 직접 대신하지는 않습니다. Android 보안 정책상 지문/PIN/패턴 잠금 해제는 시스템 잠금화면이 담당하고, 이 앱은 `showWhenLocked` Activity와 전체화면 알림으로 그 위에 할일 화면을 띄웁니다.

## 기능

- 화면이 켜질 때 할일 잠금화면 표시
- 잠금화면에서 할일 추가, 완료 체크, 삭제
- 앱 기본 화면에서도 같은 할일 목록 관리
- 부팅 후 foreground service 재시작
- 재부팅 직후 첫 지문 해제 전에도 접근 가능한 device-protected storage 사용
- 지문/PIN 해제를 계속 진행하는 버튼 제공

## 사용 방법

1. Android Studio에서 이 폴더를 엽니다.
2. `app`을 Galaxy 기기에 설치합니다.
3. 앱을 한 번 실행해서 알림 권한을 허용합니다.
4. 앱 안의 `알림/전체화면 권한 열기`에서 알림과 전체화면 알림 권한이 허용되어 있는지 확인합니다.
5. 삼성 One UI의 배터리 제한 때문에 화면 켜짐 감지가 끊기면 `배터리 최적화 예외 설정`에서 이 앱을 제한 없음으로 바꿉니다.

USB 디버깅이 켜진 Galaxy가 연결되어 있으면 PowerShell에서 아래 명령으로 설치하고 바로 실행할 수 있습니다.

```powershell
.\install-debug.ps1
```

설치 후 잠금화면 동작을 점검할 때는 아래 스크립트가 APK 설치, 앱 실행, 기본 알림 권한 부여를 한 번에 처리합니다.
앱에서 할일을 추가한 뒤 Enter를 누르면 화면 끄기/켜기를 자동으로 시도하고 `LockActivity`가 표시됐는지도 점검합니다.

```powershell
.\verify-device.ps1
```

할일 저장 포맷만 빠르게 점검하려면 JVM self-test를 실행할 수 있습니다.

```powershell
.\test-todo-codec.ps1
```

## 주의

- 일반 앱은 Android의 기본 잠금화면 자체를 완전히 대체하거나 지문인식을 가로챌 수 없습니다.
- 일부 Galaxy/Android 버전에서는 전체화면 알림 권한, 알림 카테고리, 배터리 최적화 설정에 따라 잠금화면 위 표시가 제한될 수 있습니다.
- 보안상 정상적인 흐름은 `Todo Lock 화면 표시 -> 지문/PIN으로 시스템 잠금 해제`입니다.
